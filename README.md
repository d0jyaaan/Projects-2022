## About <a name = "about"></a>
<p>
  This is a quadratic function calculator that finds:
  <ul>
    <li>Roots</li>
    <li>Y-intercept
    <li>Vertex Form</li>
    <li>Slope</li>
    <li>Min / Max Point</li>
  </ul>
  
  It also graphs the quadratic function and the slope
  <br>
